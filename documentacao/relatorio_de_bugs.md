# Relatório de Bugs

## BUG-01 — Exclusão de curso não funciona

Descrição:
Ao excluir um curso, o sistema exibe mensagem de sucesso, porém o curso continua na listagem.

Severidade: Alta  
Status: Aberto

---

## BUG-02 — Cadastro de curso vazio permitido

Descrição:
O sistema permite cadastrar cursos sem preencher campos obrigatórios.

Severidade: Alta  
Status: Aberto

---

## BUG-03 — Falta confirmação de exclusão

Descrição:
O sistema exclui cursos imediatamente sem solicitar confirmação do usuário.

Severidade: Média  
Status: Aberto

---

## BUG-04 — Ausência de botão para cancelar cadastro

Descrição:
A tela de cadastro não possui opção de voltar ou cancelar.

Severidade: Baixa  
Status: Aberto

---

## BUG-05 — Interface não informa ausência de cursos

Descrição:
Quando não existem cursos cadastrados, o sistema apenas exibe uma lista vazia.

Severidade: Baixa  
Status: Aberto
