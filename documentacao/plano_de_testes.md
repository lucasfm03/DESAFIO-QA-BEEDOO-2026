# Plano de Testes

## Objetivo
Garantir a qualidade do sistema de gerenciamento de cursos, identificando falhas funcionais, problemas de interface e ausência de validações.

## Escopo
Foram testadas as seguintes funcionalidades:

- Cadastro de cursos
- Listagem de cursos
- Exclusão de cursos
- Navegação entre telas
- Validação de campos

## Tipos de Teste Executados

- Testes Funcionais
- Testes de Validação
- Testes de Interface (UI)
- Testes de Experiência do Usuário (UX)
- Testes de Robustez

## Ambiente de Testes

Navegador: Google Chrome  
Sistema Operacional: Windows  
Ambiente: Aplicação web local

## Critérios de Sucesso

O sistema deve:

- impedir cadastro com campos obrigatórios vazios
- validar entradas do usuário
- permitir exclusão correta de cursos
- apresentar mensagens claras ao usuário
